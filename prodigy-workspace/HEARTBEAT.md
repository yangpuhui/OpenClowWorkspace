# HEARTBEAT.md - CodeProdigy 每日检查清单

## 每日检查（每次心跳触发时）

### 1. 项目进度检查
- 当前项目是什么？（查看 PROJECT-TARGET.md）
- 今天的目标完成了吗？
- 是否需要调整策略？

### 2. GitHub状态检查
- 今日新增star数量
- 最新issue和PR处理状态
- 是否有用户需要回复？

### 3. 待办事项（从 PROJECT-TARGET.md 同步）
- [ ] 需求调研完成
- [ ] 方案设计完成
- [ ] MVP代码完成
- [ ] 单元测试完成
- [ ] README文档完成
- [ ] 性能优化完成
- [ ] GitHub发布
- [ ] 社区推广

## 如果遇到卡点

### 需求挖不出
→ 换个社区，换个关键词，看看竞品

### 代码写不完
→ 砍掉MVP范围，先做核心功能

### README写不好
→ 看看同类项目的README，模仿好的结构

### star没增长
→ 去社区主动分享，不是等待被看见
