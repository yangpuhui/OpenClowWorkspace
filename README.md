# 工作区说明

## 📁 文件夹结构

### Java学习文档/
存放所有Java学习文档，按模块和知识点分类：
- 命名规范
- 常量与变量
- 集合使用
- 异常处理
- 日志规范
- Spring Boot底层
- Spring Cloud微服务
- 高并发开发

**文档命名规则**：`Java学习-{模块}-{知识点}.md`

---

### blog/
存放每日自动生成的技术博客：
- AI技术速递
- Java后端学习查漏补缺
- 明日学习预告

**博客命名规则**：`YYYY-MM-DD-{主题}.md`

---
